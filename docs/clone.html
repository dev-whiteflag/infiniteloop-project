Processo para Sync

Baixe o Github para Desktop e crie sua conta do Github, se ainda não tiver uma.

Na pagina do repo, clique no botão "Clone or Download". Depois clique no botão, "Open in Desktop".

Clique em "Clone" dentro do Github Desktop. Depois disso aperte "Fetch Origin" para Sincronizar com o repo.

Agora, o projeto deve aparecer na Aba "My Projects" dentro de seu Epic Games Launcher.

Toda vez que fizer uma modificação no Projeto e salvar, escreva o titulo da sua modificação e sua descrição, no final, aperte "Commit to master".

Depois aperte o botão "Push Origin" para sincronizar suas modificações com a repo.
