Projeto Darkpath

Darkpath é um jogo demo sendo desenvolvido por estudantes do curso de iGames da Infinite Loop.

Versionamento

O Projeto faz uso do estilo de Versionamento proposto pela Semantic.

0

Principais ou Maiores modificações no projeto, essas modificações podem quebrar o funcionamento de versões passadas, ou são tão grandes que completamente muda o funcionamento.

2

Menores ou Secundárias modificações, que podem funcionar em versões antigas ou não são grandes o bastante para mudar completamente o funcionamento do projeto a ponto de ser considerada uma Principal.

2

Patches ou Correções são modificações que corrigem problemas causados por modificações menores ou principais, não modificam o funcionamento do projeto. Geralmente essas modificações são as de maior numero.
